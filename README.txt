#Installation

Make sure that you have chrome installed
Create a folder in Desktop called "instabot"
Not paste env.prop and INSTABOT in that folder you have created
Open env.prop in notepad and enter your username (email) and password
Mention your msg and person you want to send and how many times you want to send


#Run
For Windows: Just double click on the INSTABOT.exe
For Linux/MAc: Make sure that you have java installed and run following command:
java -jar instabot_v2.jar
